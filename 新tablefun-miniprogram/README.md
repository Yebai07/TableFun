# 桌趣 - 剧本杀桌游微信小程序

## 项目简介

"桌趣"是一款剧本杀桌游微信小程序，用户可以在线体验剧本杀游戏，支持创建房间、选择剧本、角色分配、在线交流等功能。

## 功能特性

### 核心功能

1. **首页**
   - 轮播图展示
   - 快捷功能入口
   - 热门剧本推荐
   - 进行中房间显示
   - 推荐房间展示
   - 最新剧本列表

2. **剧本库**
   - 剧本搜索功能
   - 类型筛选（恐怖、推理、悬疑、古风等）
   - 难度筛选
   - 人数筛选
   - 价格筛选
   - 多种排序方式
   - 剧本详情查看

3. **剧本详情**
   - 剧本封面和信息
   - 评分和热度统计
   - 剧本简介和故事背景
   - 角色介绍
   - 亮点特色
   - 适合人群
   - 收藏功能
   - 分享功能

4. **创建房间**
   - 剧本选择
   - 房间信息设置（名称、密码、说明）
   - 游戏设置（公开、语音、自动开始）
   - 角色分配

5. **游戏房间**
   - 玩家列表管理
   - 角色分配和选择
   - 游戏阶段管理
   - 剧本内容阅读
   - 线索卡系统
   - 投票系统
   - 游戏结果展示
   - 在线聊天（文字/语音）

6. **我的收藏**
   - 用户信息展示
   - 统计数据（收藏、玩过、房间数）
   - 功能入口（我的房间、游戏记录、成就、设置）
   - 收藏剧本管理

7. **个人中心**
   - 微信登录
   - 用户信息展示
   - 等级和经验系统
   - 统计数据
   - 功能菜单
   - 退出登录

## 技术栈

- 微信小程序原生框架
- WXML - 结构层
- WXSS - 样式层
- JavaScript - 逻辑层
- 微信云开发（可选）

## 开始使用

### 前置要求

1. 安装微信开发者工具
2. 注册微信小程序账号

### 开发步骤

1. 打开微信开发者工具
2. 导入项目，选择 `tablefun-miniprogram` 目录
3. 填入小程序 AppID（可以使用测试号）
4. 点击编译即可运行

## 注意事项

1. **图片资源**：当前使用最小化PNG占位符（67字节），实际项目中需要替换为真实图片
2. **数据存储**：当前使用本地存储模拟，实际项目需要接入后端API
3. **实时通讯**：游戏房间需要使用WebSocket实现实时通讯
4. **用户认证**：需要配置微信登录功能
